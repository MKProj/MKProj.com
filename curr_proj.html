Open Source?

Open source projects are all around us, whether you are accessing a website whose server is running of a linux distro, or streaming using OBS, watching movies on plex. What does open source mean? Well the general idea of it is publicly showing a program's code, and being able to use or modify the code for anyones personal use. The usage of an open source project do depend on the open source license they are using, in the case of MKProjects, I use the General Public License version 2 or GPLv2.
